# Vue

### Project setup
```
npm install
```

### Project deploy
```
npm run deploy
```
### SRC 파일구조
```
* __test__ : jest 테스트 파일
* api 
  - coinPrice : 코인 가격 api (코인 파프리카)
  - qrCreate : 결제 SDK
* assets : 이미지 파일등...
* components
  - coin : 코인리스트, 결제 컴포넌트
  - common : 공통 컴포넌트
  - error : 404 컴포넌트
  - index : landing 컴포넌트
* hooks : hook 파일
* page : 각각의 최상위 페이지
* recoil : recoil 파일
* styles : scss 파일
* store : Vuex
* types : typescript type 파일
* utils : util js 
```
